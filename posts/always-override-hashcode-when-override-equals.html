Java：为什么重写 equals 时必须同时重写 hashCode？

2024年03月12日 计算机

Object 类是所有类的父类，hashCode 与 equals 是 Object 类中定义的方法。

如下是

package java.lang;

public class Object {
    public native int hashCode();

    public boolean equals(Object obj) {
        return (this == obj);
    }
}
