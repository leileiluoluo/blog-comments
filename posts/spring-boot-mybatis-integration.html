Spring Boot 如何集成 MyBatis 进行数据库访问？

2024年03月17日 计算机

MyBatis 是一个适用于 Java 语言的持久层框架。MyBatis 支持以注解或 XML 配置的方式来定义 SQL 查询，以及查询结果和 Java 对象的映射。MyBatis 相比于 Java 另一个流行的持久层框架 JPA 来说（具体使用请参看：「如何使用 Spring Data JPA 进行数据库访问？ 」），最大的特点是 MyBatis 具有更灵活的 SQL 控制能力。
